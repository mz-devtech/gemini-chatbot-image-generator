## 📖 Overview

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/mz-devtech/gemini-chatbot-image-generator.git
```

### 2. Navigate into the Project

```bash
cd gemini-chatbot-image-generator
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a file named:

```text
.env.local
```

Add your API key:

```env
GEMINI_API_KEY=YOUR_API_KEY_HERE
```

> Replace `YOUR_API_KEY_HERE` with your actual Google Gemini API key.

### 5. Run the Development Server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
