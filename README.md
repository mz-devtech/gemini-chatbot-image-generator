# 🤖 Gemini AI Chatbot & AI Image Generator

An AI-powered Full Stack web application built with **Next.js, React.js, Node.js, Express.js, MongoDB, and Google Gemini AI**. The application allows users to interact with an intelligent chatbot, generate AI images from text prompts, and enjoy a modern, responsive user experience.

---

## 🚀 Features

- 💬 AI Chatbot powered by Google Gemini
- 🎨 AI Image Generation
- 🔐 Secure Backend APIs with Express.js
- 📦 MongoDB Database Integration
- ⚡ Fast and Responsive UI
- 📱 Fully Mobile Responsive
- 🌙 Modern Clean Interface
- 🔄 Real-time AI Responses
- 🔒 Environment Variable Support
- 🚀 Production Ready

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/mz-devtech/gemini-chatbot-image-generator.git
```

## Go to Project

```bash
cd gemini-chatbot-image-generator
```

## Install Frontend

```bash
cd client
npm install
```

## Install Backend

```bash
cd ../server
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

GEMINI_API_KEY=your_google_gemini_api_key
```

---

# ▶️ Run the Application

## Start Backend

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm run dev
```

Frontend

```
http://localhost:3000
```

Backend

```
http://localhost:5000
```
